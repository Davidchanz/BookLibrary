# Library GUI
---

**Functional**
- Manage your Books in one place
- Share books amongs **Book Shelfs**
- Add Books from **Directory**
- Add Books from **All Sub Directory** by scpecify Root Directory
- Get Book's **Cover**
- Open and Read Book from library
